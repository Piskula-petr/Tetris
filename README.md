# Tetris

<img src="tetris.jpg" height="500">

### Description
Endless version of legendary game Tetris from 1984 build in Java.

### Technology
Java (jdk-13.0.1) <br>
Swing library

### Controls
Left arrow : Move Left <br>
Right arrow : Move Right <br>
Up arrow : Rotate <br>
Down arrow : Soft Drop <br>
