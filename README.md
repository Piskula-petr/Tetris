# Tetris

<img src="tetris.jpg" height="500">

## Description
Endless version of legendary game Tetris from 1984 build in Java. <br><br>

## Technology
Java SE <br>
Swing library <br><br>

## Controls
Left arrow : Move Left <br>
Right arrow : Move Right <br>
Up arrow : Rotate <br>
Down arrow : Soft Drop <br>
